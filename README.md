# clean_DNA_fasta
Program to clean DNA fasta sequences and removes and illegal characters in sequence with "Ns"

# Usage 
```
$ bash clean_DNA_fasta.sh <inputfile>
```
# Or  
```
$ bash clean_DNA_fasta.sh 
```
